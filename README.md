**Meraki-Api-With-Express_Mysql**

**Installation**:-
   
   
     1.Install MySQL:-


          sudo apt-get install mysql-server
     
     
    2.Install nodemon:-
    
    
         npm install --save -dev nodemon


     3.Install knex:-
         
         
         npm install knex --save
